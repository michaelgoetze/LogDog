LogDog
Logger For Dominion.Games

LogDog is an open-source Chrome extension that will keep track of your games on Dominion.Games. Everytime you play a game of Dominion, LogDog will constantly keep track of the Log for you and store it locally in Chrome (the logs are not synced between computers!).

Now you can come back to your logs anytime in a manner of seconds and save them on your computer to do some fancy analysis yourself or copy the log to use it in other apps like Windominion.com.

Just click on the LogDog logo next to your adress bar on the right, when you are on Dominion.Games

This tool is intended for post-game analysis and does not provide any real-time analysis information.

Settings

 Do not save games against Bots

Backup

Uninstall

If you want to uninstall this extension, go to chrome://extensions and remove this extension. Be aware that by removing this extension, all logs that have not been saved by downloading, will be lost permanently.

The source code for this extension is hosted on GitHub
